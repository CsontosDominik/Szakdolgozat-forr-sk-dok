A Microsoft Excel alapjai

A kurzus célja a Microsoft Excel táblázatkezelő program általános használatának elsajátítása a mindennapi életben előforduló tevékenységekhez.

 

A kurzus tartalma:

  • Adatok beillesztése
  • Cellák, oszlopok, sorok formázása
  • Cellák egyesítése
  • Szum, átlag, darabszám, maximum, minimum függvények használata
  • Rendezés
  • Szűrés
  • Diagram készítés

A kurzusomat a Microsoft Word kurzus alapján szeretném folytatni mert a két program alap logikája és felépítése nagyon hasonló.
